# Portfolio

## 🛠️ setup
1. Clone the repository and cd into `angel-mora-portfolio`.
2. Run `npm install`.

## ⚡️ Running the page

## 🤖 Development Mode:
Run `npm run dev`.
   
## 🌐 Production Mode:
When new updates are made to the website, run `npm run deploy`.
The portfolio can be found at `https://kmbappe07.github.io/angel-mora-portfolio`.
